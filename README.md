# AI-Based Student Performance & Productivity Analysis System

## 📌 Overview
This project predicts student performance using Machine Learning.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Features
- Data analysis using Pandas
- Visualization using Matplotlib
- Prediction using Logistic Regression

## 🚀 Output
Predicts whether student performance is Good (1) or Bad (0)

## 📁 Files
- main.py → main ML model
- project.py → additional code
